# Project Name
Capture The Flag

## Technologies Used
HTML,CSS,JavaScript

## Description
A game that 2 players can play, in each member turn the player will be able to select area or a line to capture, until he completed a square of 4 borders all from his own, then he will capture this box. The most player capture more boxes he will win the game. 

## User Stories

[-] As a player you will get a menu with 2 options ["Start New Game", "Load Previous Game"]
[-] As a player you will be able to click on a line to select your border claim
[-] As a player you will be able to claim a box and it will have a flag in the middle when it's claimed
[-] Each player will have a uniqe color
[-] Players will be able to save the game even if not completed in browser localestorage
[-] Players will be able to reset the game
[-] One player wins
